# Climate Challenge Week 0

## 📌 Project Overview
This project analyzes historical climate data from five African countries (Ethiopia, Kenya, Sudan, Tanzania, and Nigeria) to support Ethiopia’s preparation for COP32.

The analysis focuses on:
- Temperature trends
- Rainfall variability
- Extreme climate events
- Climate vulnerability ranking

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/your-username/climate-challenge-week0.git

2. Navigate into the project:
   cd climate-challenge-week0

3. Create virtual environment:
   python -m venv venv

4. Activate environment:
   - Windows: venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate

5. Install dependencies:
   pip install -r requirements.txt

## 📊 Project Structure

- notebooks/ → EDA and analysis notebooks
- app/ → Streamlit dashboard (optional)
- reports/ → Final report (PDF)
- data/ → ignored (not uploaded)

## 📈 Key Insights

- Rising temperature trends across all countries
- Increasing rainfall variability
- Higher frequency of extreme heat events
- Ethiopia shows growing drought vulnerability
